10 ## This program works by either linking or copying any2any to a file
11 ## which specifies the desired input and output formats by its name.
12 ## Alternately, the -i and -o command line options can be used to specify
13 ## the input and output file formats.
14 ##
15 ## For example, if you want to convert from WMV to MP4 then you could
16 ## create a symbolic link from any2any to wmv2mp4 as follows:
17 ## ln -s any2any wmv2mp4
18 ## Similarly, symbolic links (or copies or hard links) could be created
19 ## to convert from any (3 lowercase letter representation) audio/video format
20 ## to any other audio/video format. Commonly used conversions include:
21 ## wmv2mkv avi2mpg wmv2mp4 and so on.
22 ##
23 ## Naming restricton: [3 lowercase letter input]2[3 lowercase letter output]
24 ## for a 7 letter name with "2" in the middle. The 3 letter prefix and suffix
25 ## must also be a filename suffix that ffmpeg recognizes as a valid
26 ## audio/video format.
27 ##
28 ## For example, the following are filenames that conform to this restriction:
29 ## ape2m4a flv2mp4 mkv2mp4 wmv2mp4 avi2mpg wmv2mkv wma2m4a
